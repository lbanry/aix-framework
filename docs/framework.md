# AIX Framework

## Core Concept

Every interaction between a human and AI is treated as a **contract**.

The contract defines:

- Intent (what outcome is desired)
- Context (what the system knows)
- Constraints (what must be true)
- Execution (how to act)
- Validation (how to verify success)

---

## AIX Loop

1. Define contract
2. Normalize context
3. Generate execution prompt
4. Execute task (AI)
5. Validate output
6. Refine or finalize

---

## Layers

- Human Layer → defines intent
- AIX Layer → structures + enforces
- AI Layer → executes
